# Learning Paths

## AI

- antigravity : https://antigravity.google/download

- find a dataset
- open it manually
- query the AI to make you a transformation function
- query the AI to make you a function which takes the output of the previously generated function and render something

## No AI

- VSCodium : https://github.com/VSCodium/vscodium/releases

- read the course, or watch the videos
- when you're arriving at the DOM chapter, don't
- go look at the Daniel's api and sdk playlist : https://www.youtube.com/playlist?list=PLRqwX-V7Uu6YxDKpFzf_2D84p0cyk4T7X
- open a few dataset to find a questions and answers
- transform the data into something manageable, you should have a huge object or an array of objects
- render the thing based on some calculations
